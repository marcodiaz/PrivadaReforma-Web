Privada Reforma - Guia Visual de Pantallas

Documento para vecinos (explicacion simple y practica).

Indice

1) Pantallas principales | 2) Pantallas administrativas/base | 3) Beneficios de la app movil

1) Pantallas principales (uso diario)

Crear usuarios

Ruta: /admin/users

Sirve para dar de alta nuevas cuentas de manera ordenada. Aqui se define si la persona sera residente, guardia o administrador, y se asigna su unidad cuando aplica. Esta pantalla es clave porque controla quien entra a cada parte de la app.

Registrar codigo QR de visita

Ruta: /app/visits

Aqui registras a un visitante y creas su acceso. Puedes elegir tiempo de validez, cantidad de usos y numero de personas. La app genera el codigo de forma automatica para que guardia valide la entrada mas rapido.

Compartir QR

Desde: /app/visits

Muestra el QR listo para enviar al visitante. Incluye boton para copiar el link de acceso y opcion para borrar el codigo si ya no se va a usar. Esto evita confusiones y mantiene el control del acceso.

Mascotas

Ruta: /app/pets

Permite publicar informacion de mascotas de la comunidad con foto y descripcion. Ayuda a que vecinos se conozcan mejor y sepan como convivir, por ejemplo si una mascota es nerviosa o muy activa. Si algo no es apropiado, se puede reportar.

Marketplace

Ruta: /app/marketplace

Es el espacio para vender o comprar entre vecinos. Puedes subir foto, precio, descripcion y forma de contacto. Tambien incluye opcion de reporte para cuidar el contenido dentro de la comunidad.

Paquetes

Ruta: /app/packages

Te deja ver el estado de tus paquetes y confirmar cuando pasaras por ellos. El flujo esta pensado para seguridad: guardia resguarda, residente confirma, guardia entrega.

Incidencias

Ruta: /app/incidents

Aqui la comunidad reporta problemas y les da prioridad con votos. Sirve para que lo mas urgente suba primero y se atienda antes.

Crear incidencia

Sobre: /app/incidents

Formulario simple para registrar el problema: titulo, descripcion, categoria y prioridad. Con eso el reporte entra al flujo de seguimiento.

Reporte de estacionamiento

Ruta: /app/parking

Si alguien ocupa un cajon que no le corresponde, aqui se reporta directo a guardia. Tambien muestra historial para saber si ya fue atendido.

Perfil (nombre y contrasena)

Ruta: /app/profile

Pantalla para actualizar tu nombre visible y tu contrasena. Es importante para mantener tus datos correctos y tu cuenta segura.

Notificaciones (admin)

Ruta: /admin/push

Permite enviar avisos a usuarios especificos, por unidad o por rol. Es util para mandar mensajes urgentes de operacion en segundos.

2) Pantallas administrativas/base

Moderacion de reportes

Ruta: /admin/reports

Cuando hay contenido reportado, aqui se revisa y se decide si se elimina o se cierra el reporte. En la imagen se muestra un estado sin reportes abiertos.

Paqueteria global (admin)

Ruta: /admin/packages

Vista global para supervisar paquetes de toda la privada por estatus. Ayuda a detectar carga operativa y retrasos.

Estado de cuenta

Ruta: /app/finance

Base de informacion financiera para residente/comite. Esta pantalla puede crecer a detalle de movimientos y transparencia.

Adeudos (admin)

Ruta: /admin/debts

Pantalla para enfoque de cobranza y morosidad. Funciona como base para ampliar controles financieros.

Finanzas (admin)

Ruta: /admin/finance

Vista ejecutiva para ingresos, egresos y presupuesto. Actualmente sirve de estructura para evolucion futura.

Exportaciones (admin)

Ruta: /admin/exports

Modulo pensado para generar reportes CSV/PDF para administracion y comite.

3) Beneficios de instalar la app movil (RN)

La app instalada mejora tiempos de respuesta y seguridad operativa:

  • Notificaciones mas confiables que en navegador movil.
  • Vibracion y sonido por tipo de alerta (SOS, estacionamiento, incidencia).
  • Temporizadores SLA para que guardia vea cuanto tiempo lleva cada caso.
  • Posible uso de ubicacion (con permiso) para contexto operativo y mejor coordinacion.
